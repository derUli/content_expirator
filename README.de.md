# Content Expirator für UliCMS
content_expirator ermöglicht, für jede Seite einen Zeitraum festzulegen, in dem die Seite aktiv ist. Wenn der Zeitraum anfängt, wird die Seite aktiviert. Wenn der Zeitraum abgelaufen ist, wird die Seite wieder deaktiviert.
Ideal geeignet z.B. für die Umsetzung von zeitbeschränkten Angeboten.

## Voraussetzungen
* [UliCMS 2018.1.1](https://en.ulicms.de/current_versions.html)
* [BetterCron 1.0](https://extend.ulicms.de/better_cron.html)
